# Délimitation du DPF - QGIS Plugin

Ce plugin, développé dans le cadre d'un projet de recherche, permet de détecter automatiquement les limites du domaine public fluvial au droit de sections transversales.

## How to install and use  

Pour toute information concernant l'installation et l'utilisation du plugin, se référer à la [notice d'utilisation du plugin](./notice_delimitation_DPF.pdf).
